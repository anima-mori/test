如果參與這個訓練，會怎麼安排學習時間？
因預期要投入訓練，已將工作調整為每週八小時的彈性遠端工作，可將大多數時間投入訓練中。多年前曾經學習過 HTML、CSS、Javascripts，預計利用開課前複習。
軟體技術日新月異，如何確定選擇投入的領域是正確有回報的？
身為一個門外漢，我認為自己對於整個技術世界的認識都很有限，因此能投入的領域選項也不多。在投入訓練的過程中，增加對軟體技術的素養，對我就是某程度的回報。例如學會串接 API 、爬蟲技術，可以用於抓取歷史股價作回測、學習了資安技術，可以在日常避免自己做出風險行為。此外，新的技術會一直出現，我在大學選修資訊相關課程時，區塊鏈、生成式 AI 這些技術都還沒問世，如果能給當時的自己建議，說真的也就只是打好基礎，然後選一個自己感興趣的領域，缺少的知識就去學習，然後跳進去長期耕耘。我覺得人生充斥隨機性和運氣，即使投入某領域被認為正確，經常都是事後諸葛。只要當下覺得是最好的決定，認真投入並且能說服自己即可。
請描述一件產生明顯負面情緒的經歷，如何處理該情緒？
工作時，若 to-do list 上超過十項以上待辦事項，我的內心活動很容易糾結於如何排程？哪些事項需要主管授權或是與同事凝聚共識後才能執行？什麼事項死線已在眼前？在這種狀態下，有時會不自覺陷入焦慮中。可以的話，我會花一點時間向主管確認一下大方向，避免偏離太遠，也確認需要優先執行的事項。跟共事者保持良好溝通，有時能避免自己在盲點上過度打轉。接著我可能會沖杯咖啡坐定下來，入地獄般的覺悟心情，一件一件事項開始執行，甚至不斷重播近期覺得順耳的歌，隨著事項減少甚至好運時會進入心流狀態，可以藉此緩解一些焦慮的情緒。
關於這份申請網頁，分享一個開發時的技術心得。
因距離學習 HTML 與 CSS 已距離多年，加上需要在短時間內生成這份網頁，因此我使用 Claude.ai，Prompt 是「請給我一個有七個 Q&A 的簡潔繁體中文 html 版型」接著替換內容。
關於網頁的 CSS：
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">此 meta tag 說明網頁是 Responsive Web Design，顯示寬度為裝置寬度，預設縮放比例為 100%。
  • @media (max-width: 600px) {body {padding: 15px;} 則是說明，當裝置寬度 < 600 px 時，body 的 padding 將由預設 20px 減少為 15 px
以前學習時沒有生成式 AI，覺得將 AI 的使用納入學習、工作的流程中，也是一件非常重要的課題。
如何看待自身工作和整個社會群體的連結關係？
覺得自身工作跟社會是個「共生／共好」的關係，除了透過工作得到某種身份與社會群體互動、甚至獲得認同；另一層關係是用工作帶來的生活餘裕去支持自己認同的理念，例如支持地震賑災的捐款、選購動物友善的商品。
